### 目标功能
1. 基本要求：
- （1）目标捕获与稳定跟踪
   用约 1.8cm 宽的黑色电工胶带在 A4 纸四边贴一个长方形，构成 A4 靶
纸。将此 A4 靶纸贴在屏幕自定的位置。系统启动后，应自动识别靶面上的目标
图形，并控制云台使指示光斑稳定对准在图形中心。光斑中心与目标几何中心距
离误差 ≤1cm，并持续保持 ≥5s。初始校准时间时间不大于 5s。（10 分）

- （2）多目标顺序打靶
系统由选手切换至连续打靶状态，靶面上放置 3～5 个不同位置的目标图
形，系统启动后，应按图形边数顺序升序（如圆形->方形->星形）依次将光斑移
动至各目标中心，并在每个目标处稳定停留 2s。每完成一个目标，进行一次声
光提示。总完成时间不大于 20s。（15 分）

- （3）指定图形绘制
系统由选手切换至绘图状态，启动运动目标控制系统，光斑能在按照上题顺
序依次沿着每个图形上的胶带顺时针移动一周。每个图形 30s，超时不得分，光
斑完全脱离胶带一次扣 2 分，连续脱离胶带移动 5cm 以上记为 0 分。（25
分）

3. 发挥部分
- （4）动态靶心追踪
靶标平面以 5-10rpm（初定）随机指定速度旋转，云台应该能稳定瞄准目标
靶心，目标靶面采用 A4 纸，用 1.8cm 宽的黑色胶带沿四周边缘贴一个长方形，
勾勒出靶纸的外轮廓；用红色油性记号笔在靶的中心标一个点作为靶心，靶心直
径≤0.1cm，并以 此为圆心分别以 2、4、6、8 和 10cm 为半径画红色圆，圆弧
线宽度≤0.1cm。Di ≤2cm（i=1，2）为满分，Di 每增加 1cm 扣 1 分，不足
1cm 按 1cm 计算。（35 分）

- （5）动态轨迹同步绘制
激光笔沿靶面上半径 6cm 的红色圆弧同步画圆，靶标平面以 5-10rpm
（初定）随机指定速度旋转。光斑与半径 6cm 的红色圆弧线最大距离Ｄ
2≤2cm；靶面转动 1 圈，正好画 1 圈光斑，同步误差必须小于 1/4 圈
